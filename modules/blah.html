CSS Backgrounds - Definition and usage

CSS background is an attribute which defines the background of elements. There are various kinds of background attributes like:

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

Note: there are a few more background attributes like: background-clip,
background-blend-mode, background-origin et cetera.. in this description
we will be focused in the common kinds of the background attribute

background-color:

The background-color attirbute specify the background color of an element.
For example:
if we will use an embedded css style to color the background of a button:
